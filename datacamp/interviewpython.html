0. Python Data Structures and String Manipulation

0-0. What are the main data structures in Python?

  • List : an ordered mutable sequence of items (e.g. numbers, strings, etc)
    • my_list = [1,2,3,4]
      print(my_list[2]) #3
      print(my_list[0:3]) #1,2,3
    • methods:
      • .append()
      • .remove()
      • .pop() #removes the last item from a list and returns its value
      • .count()
      • my_list.append(10)
        my_list.remove(10)
        my_list.pop() #3
  • Tuple : an ordered immutable sequence of items
    • my_tuple = (1, 'apple', 2, 'banana')
  • Set : an unordered collection with no duplicate items
    • my_set = set([1,2,3,4,5])
      print(my_set) #{1,2,3,4,5}
      my_set = set([1,1,1,2,2,2,3,4,4,5])
      print(my_set) #{1,2,3,4,5}
    • methods:
      • .add()
      • .remove()
      • .union()
        • my_set1.union(my_set2)
      • .intersection() : returns a new set with only common items
        • my_set1.intersection(my_set2)
      • .difference()
        • my_set1.difference(my_set2)
  • Dictionary: a collection of key-value pairs where keys are unique and immutable
    • fruits = {'apple':10, 'oragne':6, 'banana':9}
      fruits = dict([('apple', 10), ('orange',6), ('banana',9)])
      fruits['apple'] #10
    • list(fruits.items()) #[('apple', 10), ('orange',6), ('banana',9)]
    • fruits.keys() #dict_keys(['apple', 'orange', 'banana'])
      fruits.values() #dict_values([10, 6, 9])
    • .popitem() #remove the last inserted key-value pair
      • fruits.popitem('banana')

0-1. What are common ways to manipulate strings?

  • __str__ constructor: when an instance is called with str(), this method is executed
    • class newClass:
        def __init__(self, num):
          self.num = num
        def __str__(self):
          return str(self.num)
      nc = newClass(3)
      str(nc) #'3'
  • .index()
    • s = "interview"
      s.index('n') #1
      s.index('i') #0
  • Strings are immutable
    • s[0] = 'a' #TypeError
  • methods: .capitalize(), .lower(), .upper(), .replace() -> return a new string object
  • .replace()
    • s1 = 'a dog ate my food'
      s2 = s1.replace('dog', 'cat')
      print(s2) #a cat ate my food
  • create a string from a list of strings
    • l = ['I', 'like', 'to', 'study']
      s = ' '.join(l) #' ' is a delimiter between strings
      print(s) #I like to study
  • breaking a string into a list of strings:
    • l = s.split(' ')
      print(l) #['I', 'like', 'to', 'study']
  • String methods with DataFrames
    • import pandas as pd
      d = {name: ['john', 'amanda', 'rick'], 'age':[35, 29, 19]}
      D = pd.DataFrame(d)
      print(D)
      D['name'] = D['name'].str.capitalize()
