Data Science for Everyone

Introduction to Data Science

0-0. What is data science?

  • data science : a set of methodologies for taking in thousands of forms of data that are available to us today, and using them to draw meaningful conclusions.
  • What can data do?
    • describe the current state of an organization of process
    • detect anomalous event ex.fraudulent purchases
    • diagnose the causes of events and behaviors
    • predict future events
  • why so popular? -> we're collecting more data than ever before.
  • The data science workflow
    1. Data Collection & Storage
    2. Data Preparation
        ex.cleaning data(finding missing&duplicate data, convert into organized format)
    3. Exploration & Visualization
        ex.building dashboard, performing comparisons
    4. Experimentation & Prediction
        ex.building a forecast, perform a test

0-1. Applications of data science

  1. Traditional Machine Learning
    • What do we need for machine learning?
      • a well-defined question
      • training data: used to build an algorithm
      • a new set of data to use our algorithm on
    • ex. detecting fraudulent transactions
  2. Internet of Things(IoT)
    • refers to gadgets that aren's standard computers
    • ex. smart watches, internet-connected home security systems, electronic toll collection systems, building energy management systems
  3. Deep learning
    • many neurons work together
    • requires much more training data
    • used in complex problems
      • ex. Image classification, language learning/understanding

0-2. Data science roles and tools

  • Data Engineer
    • focus on data collection & storage
    • tools
      • SQL: to store and organize data.
      • Java, Scalar, or Python: to process data
      • Shell: command line to automate and run tasks
      • Cloud computing: AWS, Azure, Google Cloud Platform. to ingest and store large amounts of data
    • Information architects
    • Build data pipelines and storage solutions
    • Maintain data access
    • design infrastructure so that data is not only collected, but easy to obtain and process
    • Build and configure SQL, storage solution
  • Data Analyst
    • focus of data preparation & Exploration & Visualization
    • tools
      • SQL: retrieve and aggregate data
      • Spreadsheets: simple analysis
      • BI tools: Tableau, PowerBI, Looker. Dashboards and visualizations
      • may have Python or R: clean and analyze data
    • perform simpler analyses that describe data
    • create reports and dashboards to summarize data
    • clean data for analysis
    • have less programming and stats experience than the other roles.
  • Data Scientist
    • focuse on Data Preparation, Exploration & Visualization, and Experimentation & Prediction
    • tools
      • SQL: retrieve and aggregate data
      • Python and/or R: data science libraries e.g.,pandas, tidyverse
    • versed in statistical methods
    • run experiments and analysies for insights
    • traditional machine learning
  • Machine Learning Scientist
    • focus on Data Preparation, Exploration & Visualization, and Experimentation & Prediction
    • tools
      • Python and/or R : machine learning libraries e.g., TensorFlow, Spark
    • similar to data scientist, but with a machine learning specialization
    • predictions and extrapolations
    • classificaiton
    • deep learning
      • Image processing
      • Natural langauge processing
